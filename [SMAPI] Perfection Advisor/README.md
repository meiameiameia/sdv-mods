# [SMAPI] Perfection Advisor

Standalone read-only completion/perfection assistant.

## v1 Slice

- Standalone summary UI entry point (`perfection_advisor_open` or configured keybind).
- Advisor is disabled by default.
- Spoiler details are disabled by default.
- Read-only progress summary only (no control/automation behavior).

## Default Controls

- Keybind: `F8`
- Console command: `perfection_advisor_open`

## Default-Off Safety

The mod does not show advisor output until `EnableAdvisor` is explicitly set to `true`.

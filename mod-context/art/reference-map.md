# Prompt To Reference Map

Use this file to run one sprite generation at a time with a locked prompt and exact upload set.

All references below are reference-only inputs, not runtime assets.

Paths under `mod-context/art/references/` are the canonical local-only locations for raw upload images. The mapping file is tracked in Git; the raw reference PNGs are intentionally ignored.
